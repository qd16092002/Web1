## Install
1. Cài đặt các gói cần thiết:
    ```sh
    npm i
    ```

2. Thêm các tệp `.env`:
    - **.env**
        ```env
        DB_URL=mongodb://localhost:27017/nhattest
        DB_URL_2=mongodb://192.168.1.17:27017/nhattest
        PORT=3002
        ACCESS_TOKEN_SECRET=ACCESS_TOKEN_SECRET
        REFRESH_TOKEN_SECRET=REFRESH_TOKEN_SECRET

3. Chạy ứng dụng:
    ```sh
    npm start
    ```
